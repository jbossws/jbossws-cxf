JBossWS

Welcome to JBoss Web Services

JBossWS is a Web Service Framework contributing to the JBoss' JavaEE5 offering. It integrates supported JAX-WS compliant web service stacks in JBoss AS while adding simplified management, configuration, tooling, etc.
